#### 简介：

基于 Vue3 + Lingo3D 的 Web 3D 游戏， 可以控制小狐狸在场景中自由转动视角，自由行走，跳跃，所有物体均有碰撞体积

#### 技术栈：

`Vue3 + Lingo 3D`

#### 安装教程：

###### 克隆项目

`git clone https://github.com/jxz200/animalCrossing.git`

###### 安装依赖

`npm install`

###### 启动项目

`npm run dev`
